Per la categoria il programma deve leggere prima tutti i libri e nel campo "categoria" inserisce un valore di default/vuoto.
Successivamente sarà l'utente e/o programmatore a modificare la categoria.
Array di struct con valori struct ovvero il "primo array" va a contenere le struct dei libri,
dentro la categoria c'è un altro array di struct dove sono contenuti tutti i libri di quella categoria specifica con il titolo, l'autore, prezzo, anno

Passaggi:
1. Creazione dell'array di struct "Libreria" per i libri, con anche quello delle categorie perchè verrà inserito un valore di default e dopo verrà modificato dall'utente
2. Lettura dei dati dal file e inserirli dentro le struct del punto 2
3. Menu per agevolare l'utilizzo
4. Inserimento dall'utente delle categorie dentro i libri
5. Smistamento dei libri dentro le loro categorie utilizzando un array a parte
